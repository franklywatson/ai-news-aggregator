Anthropic Backtracks on Claude Agent SDK Pricing Overhaul

Anthropic abruptly paused plans to shift Claude Agent SDK usage to costly API-based billing, a move that would have severely increased expenses for power users and third-party apps. The release serves as a reference architecture for building multi-model agent harnesses with built-in toolkits and MCP support.

GitHub highlights how AI-driven development has popularized the use of Git worktrees to support parallel coding sessions. The GitHub Copilot app now defaults to using worktrees, allowing human developers and agents to work on multiple branches simultaneously without disrupting local environments.

AI Models & Research

Study Reveals Claude Code is 98% Traditional Software Engineering

A deep architectural analysis of Claude Code's source code reveals that the system is 98% traditional software scaffolding and 2% AI model. The study highlights how the true complexity of agentic systems lies in the harness—permission layers, context compaction, and subagent delegation—rather than the core inference loop.

"This empirically validates the 'big harness, small model' thesis, proving that engineering surrounding the model is currently the primary bottleneck and competitive moat in agent development."

OpenAI Unveils 'Deployment Simulation' for Pre-Release Safety

OpenAI has introduced Deployment Simulation, a new methodology using real conversation data to predict how AI models will behave in the wild before they are officially released. The technique aims to improve the accuracy of safety evaluations and catch emergent behaviors early.

"As agents gain autonomous capabilities, simulating deployment environments becomes critical for anticipating coordination failures, security vulnerabilities, and safety regressions."
