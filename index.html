AI Models & Research

ParallelKernelBench Exposes LLMs' Weakness in Multi-GPU Code Generation

Together.ai introduced ParallelKernelBench (PKB), a benchmark evaluating how well frontier LLMs can write optimized multi-GPU CUDA kernels across 87 real-world workloads. While the best models solved less than a third of the problems zero-shot, agentic feedback loops provided only modest improvements, highlighting a significant gap in AI reasoning for complex distributed coordination.

"As AI models increasingly attempt to autonomously manage infrastructure and optimize their own training loops, their inability to reason about rank coordination and interconnect bottlenecks represents a critical barrier to fully autonomous AI R&D."

Datasette 1.0a35 Introduces Browser-Based Database Management

Simon Willison released Datasette 1.0a35, adding robust JSON APIs for creating and altering SQLite tables directly within the browser using Pyodide and WebAssembly. The release also leverages the Origin Private File System (OPFS) to allow persistent, local file editing via web-based agents.

"Bringing powerful database management entirely to the client side opens up new possibilities for local-first AI applications and privacy-preserving agent workflows."
