AI Agents Collaborate to Solve Century-Old Math Problems on EinsteinArena

EinsteinArena is a new platform where AI agents collaborate and compete on open mathematical problems, achieving 11 new state-of-the-art results including pushing the kissing number lower bound in dimension 11 from 593 to 604. The breakthrough demonstrates that multi-agent coordination, where agents share partial results and build on each other's work, can solve problems no single agent could solve alone.

"This is a concrete proof point that collaborative, multi-agent search can accelerate scientific discovery in production environments, moving beyond isolated agent benchmarks toward real emergent coordination."

Aurora: Self-Improving Speculative Decoding That Learns From Live Traffic

Aurora is an open-source RL framework that continuously updates a speculative decoding draft model from live inference traces, turning the system into a self-improving flywheel that adapts to shifting traffic patterns without service interruption. It achieves a 1.25x additional speedup over well-trained static speculators, and online training from scratch can even outperform carefully pretrained baselines.

"By closing the loop between serving and training, Aurora removes the costly offline pipeline maintenance burden and ensures inference acceleration compounds over time as real usage data accumulates."

Mamba-3: State Space Models Redesigned for Inference-Heavy Agentic Workloads

Mamba-3 is a new state space model architecture designed with inference efficiency as the primary goal, featuring more expressive recurrence, complex-valued state tracking, and a MIMO variant that boosts accuracy without slowing decoding. The 1.5B SISO model achieves the fastest prefill+decode latency across all sequence lengths compared to Transformers and prior linear alternatives.

"As agentic workflows like Claude Code and Codex push inference demand to new heights, architectures explicitly built for decode efficiency over training speed become critical for sustainable deployment economics."
